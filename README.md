# 🏥 Hospital Chatbot System

A Hospital Chatbot application developed using **Python** and **Streamlit** that provides secure patient registration, login authentication, appointment booking, and automated hospital assistance.

---

# 🚀 Features

- Patient Registration System
- Secure Login Authentication
- JWT Token Authentication
- Appointment Booking
- Hospital Information Chatbot
- Emergency Assistance Responses
- SQLite Database Support
- Optional MySQL Database Support
- Streamlit Interactive UI
- Password Encryption using bcrypt

---

# 🛠️ Technologies Used

- Python
- Streamlit
- SQLite
- MySQL
- JWT Authentication
- bcrypt
- Git & GitHub
